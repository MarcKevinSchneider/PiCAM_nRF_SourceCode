# PiCAM_nRF_SourceCode
Source code for the XIAO nRF52840 microcontroller for the new PiCAM model. Code was written as part of the final project of the "Proximate Sensing of Biodiversity" course of the Environmental Informatics Department of the Philipps-University Marburg.

For the repository of the Raspberry Pi Zero 2W source code, click [here](https://github.com/MarcKevinSchneider/PiCAM_PiZero2W_SourceCode).
